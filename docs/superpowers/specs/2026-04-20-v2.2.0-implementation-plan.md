- `TaskEntity` - 任务实体

#### Step 1.4 REST API实现
| 端点 | 实现内容 |
|------|----------|
| `/api/tasks` | 任务列表（GET） |
| `/api/tasks/stats` | 统计数据（GET） |
| `/api/tasks/{id}` | 任务详情（GET） |
| `/api/tasks/{id}/pause` | 暂停（POST） |
| `/api/tasks/{id}/resume` | 恢复（POST） |

| `/api/tasks/cancel/{id}` | 取消（POST） |

- `SseController` - SSE端点 `/api/events/stream`
- 事件类型：`progress_update`, `tool_call`, `intervention_prompt`
- 心跳机制（30秒）

#### Step 1.6 定时清理任务
- `CleanupScheduler` - 每日03:00执行
- 清理7天前已完成/已取消任务

#### Step 3.3 StepScroller组件
**关键实现**：
- 横向滚动容器
- 鼠标拖拽滚动
- 小手光标切换（grab/grabbing）
- 触摸滑动支持

```typescript
// 核心拖拽逻辑
const startDrag = (e: MouseEvent) => {
  isDragging = true;
  startX = e.pageX - scrollerRef.scrollLeft;
  scrollerRef.style.cursor = 'grabbing';
};

const onDrag = (e: MouseEvent) => {
  if (!isDragging) return;
  e.preventDefault();
  scrollerRef.scrollLeft = e.pageX - startX;
};

const endDrag = () => {
  isDragging = false;
  scrollerRef.style.cursor = 'grab';
};
```
